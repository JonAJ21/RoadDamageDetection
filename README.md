# Лабораторная работа по курсу "Киберфизические системы".

Выполнил: Киселев Артём Олегович

Группа: М8О-403Б-22

Работа выполнена в файле rdd.yaml

Для запуска необходимо скачать датасет с kaggle:

https://www.kaggle.com/datasets/aliabdelmenam/rdd-2022/data

Распаковать скаченный архив в корневую папку репозитория.

Подготовить окружение с помощью uv

Запустить rdd.ipynb


Результатами запусков программы соответствуют:
1) baseline_yolov11n - runs/detect/val-6, runs/detect/test1/baseline_yolo11n

2) improved_baseline_yolov11n - runs/detect/val, runs/detect/test1/improved_baseline_yolo11n

3) baseline_custom - runs/detect/val-8, runs/detect/test1/custom_model_baseline-5

4) improved_baseline_custom - runs/detect/val-10, runs/detect/test1/improved_baseline_custom
